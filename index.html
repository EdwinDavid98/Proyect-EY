Funciones del Asistente

Este asistente puede realizar tareas como:

  • Cálculos matemáticos básicos
  • Manipulación de texto
  • Integración con una API de lenguaje natural

Implementación del Cálculo Matemático

Para realizar cálculos matemáticos básicos, el asistente utiliza una interfaz gráfica desarrollada con tkinter. Además, se implementa una validación mediante expresiones regulares para asegurar que la expresión ingresada por el usuario solo contenga números, operadores básicos y paréntesis. Esto evita errores y muestra una advertencia si se introducen caracteres no permitidos, garantizando la seguridad y precisión de los cálculos.
